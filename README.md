# react-psh-image-editor-modal

A reusable React modal component to crop, resize, scale, and export images in multiple formats.

## Features
- Crop & zoom images (`react-easy-crop`)
- Resize images manually or via **social media presets**
- Export as `.png`, `.jpg`, `.jpeg`, `.webp`
- Preview & download edited image
- Reusable modal for multiple projects (e-commerce, social media, newsletters)
